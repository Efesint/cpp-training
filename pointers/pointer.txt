2. Указатель (Pointer)

Указатель — это переменная, которая не хранит данные напрямую. Вместо этого она хранит адрес какой-то другой переменной.
То есть, это бумажка, на которой записан номер квартиры.

    Как объявить? Используй звездочку * после типа данных.

        int* ptr; — «переменная ptr, которая может хранить адрес ячейки типа int».

        double* dPtr; — «бумажка для адреса типа double».

        Важно: Указатель имеет тип! Указатель на int (int*) не может просто так хранить адрес переменной типа double. Бумажка для обычной квартиры не подойдет для пентхауса.

    Как присвоить адрес? Сохрани адрес, полученный с помощью &.

        int* ptr = &x; — теперь ptr «указывает» на x.

    Как получить значение по адресу? Используй оператор разыменования * перед именем указателя. Это действие говорит: «Пойди по адресу, записанному на этой бумажке, и возьми то, что там лежит».

        int x = 10;

        int* ptr = &x;

        cout << *ptr; — выведет 10. Мы пошли по адресу в ptr и нашли там x.

        *ptr = 20; — пойди по адресу в ptr и положи туда 20. Теперь x тоже стал 20!

    Ключевой момент: Символ * используется в ДВУХ контекстах. Не путай!

        При объявлении: int* ptr; — «я создаю переменную-указатель».

        При использовании: *ptr = 5; — «я работаю со значением ПО АДРЕСУ, который хранится в ptr».
